安装：（这里假设你是ROOT）

安装jdk到/usr/local/目录下

cd /usr/local
tar zxf IBMJava2-13.tgz
    #这里要把IBMJava2-13.tgz的全路径名加上
ln -s IBMJava2-13 jdk
    #做软链接，以后使用JDK时只需要指定目录/usr/local/jdk

安装Jikes到/usr/local/目录下

cd /tmp
tar zxf Jikes1.2.tar.gz
cd jikes1.2
./configure
make && make install
    #要耐心等待，Jike的编译速度比较慢... ...
    #缺省jikes会安装到/usr/local/bin下面。

安装Resin到/usr/local/目录下：

cd /usr/local
tar zxf resin-1.2.b2.tar.gz
    #需要加上全路径名
ln -s resin-1.2.b2 resin
    #做软链接，以后使用resin只需要指定目录/usr/local/resin
cd resin
export JAVA_HOME=/usr/local/jdk
    #制定jdk的路径
./configure
    #我们这里不配置Apache，直接使用Resin自带的Web服务器
make
    #编译
make install
    #安装

测试Resin Server：

resin编译完毕，就已经可以用了。其实make install只有需要和Apache配合使用的时候才需要做。

那么下一步，我们来启动resin的webserver。

cd /usr/local/resin
bin/httpd.sh start
    #好了，我们的Resin服务器应该已经启动了... 嗯，如果正常的话。

启动一个浏览器，在URL中输入：http://ur-resin-host:8080/看看是否有Resin的首页？

注：resin缺省启动监听在8080端口上。

如果正常，那么我们可以进入下一步的配置。

注：

如果你不打算直接使用resin自带的webserver，那么你可以在configure resin的时候加上参数：--with-apache=DIR the Apache root directory,
之后当你make && make install的时候，resin会自动帮助你修改apache/conf/httpd.conf，增加对resin模组的加载。
